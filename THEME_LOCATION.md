## Troubleshooting

**Can't see the theme controls?**
- Make sure you're on the templates page (`/dashboard/templates`)
- The sidebar must be visible (desktop view or expanded on mobile)
- Scroll down in the sidebar if needed

**Theme not changing?**
- Check browser console for errors
- Make sure JavaScript is enabled
- Try refreshing the page

**Dropdown not appearing?**
- Click the palette icon (ğŸ¨) button
- Make sure you're clicking the right button (second one)
